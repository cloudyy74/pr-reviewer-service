# pr-reviewer-service
Сервис для автоматического назначения ревьюеров на Pull Request’ы, управления командами и участниками, а также получения информации о назначенных ревью.

## Инструкция по запуску

### Требования

Необходимо чтобы был установлен `Docker` и `docker-compose`

### Запуск

1. Создать `.env` файл с этими переменными (можно командой `make env`) и убедиться что они согласованы с использующимся конфигом в `./config/`
```dotenv
POSTGRES_DB=...
POSTGRES_USER=...
POSTGRES_PASSWORD=...
```

2. Запуск контейнера с приложением и всем необходимым окружением:
```commandline
docker-compose up
```
